# 뮤피(Mupy) AI 어시스턴트 작업 저장소

뮤피(Mupy)는 AI 어시스턴트가 수행한 작업 결과물을 체계적으로 관리하는 저장소입니다. 각 프로젝트는 독립된 폴더로 관리되며, 작업 메타정보와 함께 결과물이 저장됩니다. 클로드 코드 등 다른 AI 에이전트가 이 저장소의 README만 읽어도 전체 작업 맥락을 파악할 수 있도록 설계되었습니다.

## 레포지토리 구조

- **README.md**: 현재 보고 계신 이 파일로, 레포지토리의 전체적인 소개와 구조, 프로젝트 목록을 안내합니다.
- **projects/**: 개별 프로젝트 결과물이 저장되는 폴더입니다. 각 하위 폴더는 하나의 독립된 프로젝트를 의미합니다.
